# Barberia CRUD System

A full-stack web application for managing a barbershop, including clients, appointments, services, and barbers.

## Features

- Create, read, update, and delete clients
- Manage appointments and schedules
- Handle barbers and services
- Full integration between frontend and backend
- Responsive user interface

##  Technologies Used

### Backend
- Node.js
- Express.js
- SQLite

### Frontend
- React
- Vite
- JavaScript (ES6)

### Tools
- Git & GitHub
- Thunder Client (API testing)
